## [0.1.0] - 2025-11-27
### Added
- Initial public release of **Personal -  AiChat**, the GPT-5.1-powered Telegram copilot with LangChain tooling.
- Modular tool suite (utilities, memories, shopping list, math agent, file generation) with automatic loader.
- Attachment ingestion pipeline for text, media, audio transcription, PDF, Word, Excel, and CSV files.
- SQLite persistence for users, chat memories, Telegram message cleanup, and long-term storage.
- Comprehensive README, MIT license, and this changelog for transparent documentation.
